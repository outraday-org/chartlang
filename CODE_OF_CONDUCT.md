# Code of Conduct

## Purpose

This project exists to build good software. This document sets expectations for how contributors interact so that collaboration stays productive and focused on the work.

## Standards

Expected behavior:

* Be respectful and professional in all communication
* Give and accept constructive, technical feedback
* Stay focused on the work and the merits of ideas
* Take responsibility for mistakes and learn from them

Unacceptable behavior:

* Personal attacks, insults, or harassment
* Off-topic disputes unrelated to the project
* Publishing others' private information without permission
* Sustained disruption of discussions, issues, or pull requests

## Scope

This Code of Conduct applies to all project spaces, including the repository, issues, pull requests, and any official communication channels.

## Enforcement

Maintainers are responsible for enforcing these standards and may remove, edit, or reject contributions that do not align with them. Serious or repeated violations may result in temporary or permanent removal from the project.
