# @invinite-org/chartlang-host-worker

`experimental`

Web Worker ScriptHost for the browser.

## Install

```bash
pnpm add @invinite-org/chartlang-host-worker
```

## Public surface

Planned (Phase 1+): `createWorkerHost() → ScriptHost`.

## Minimum-viable API call

```ts
import { PACKAGE_VERSION } from "@invinite-org/chartlang-host-worker";
console.log(PACKAGE_VERSION); // "0.0.0"
```
