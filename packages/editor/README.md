# @invinite-org/chartlang-editor

`experimental`

CodeMirror 6 reference editor over @invinite-org/chartlang-language-service.

## Install

```bash
pnpm add @invinite-org/chartlang-editor
```

## Public surface

Planned (Phase 1+): `createChartlangEditor(opts)`, `<ChartlangEditor />` React component.

## Minimum-viable API call

```ts
import { PACKAGE_VERSION } from "@invinite-org/chartlang-editor";
console.log(PACKAGE_VERSION); // "0.0.0"
```
