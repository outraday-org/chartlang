# Script manifest

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §4.1 / §5.

The `ScriptManifest` schema: the JSON sidecar the compiler emits
alongside `.chart.js`, declaring the script's `apiVersion`, inputs
schema, declared capabilities, plot / draw / alert kinds in use, and
the description / metadata fields hosts surface in UI.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the Phase 1 compiler PR.
