# Grammar (TS subset)

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §5 / §17.3.

The canonical grammar of chartlang: the TypeScript subset the compiler
accepts, the `defineIndicator` shape it expects, and the static
analyses that turn forbidden constructs into compile-time errors.
Pine authors looking for source-to-source equivalents should start
with the [Pine to chartlang migration guide](./pine-migration.md).
This document is part of the canonical language spec — it must be
self-contained, with no "see the source code" references.
