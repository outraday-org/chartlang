# Embed in your chart

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §22.9 PR 11.

A guide for chart-app authors who want to run user-supplied chartlang
scripts inside an existing chart UI: wiring `@invinite-org/chartlang-host-worker`,
implementing the `Adapter` interface, declaring capabilities, and
piping Plot / Draw / Alert emissions into the chart's render layer.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the PR 11 reference embed in Phase 1.
