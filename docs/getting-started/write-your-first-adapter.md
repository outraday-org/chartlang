# Write your first adapter

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §17.4.

A walkthrough for adapter authors: scaffolding with
`@invinite-org/chartlang-adapter-kit`, declaring the `Capabilities`
surface, implementing the `Adapter` interface for plot / draw / alert
emissions, and proving correctness against the
`@invinite-org/chartlang-conformance` suite.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the Phase 1 adapter-kit PR.
