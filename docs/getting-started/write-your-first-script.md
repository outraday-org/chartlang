# Write your first script

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §19 Phase 1.

A walkthrough that takes a new contributor from a clean clone to a running
EMA-cross indicator: install `@invinite-org/chartlang-core`, scaffold a
`.chart.ts` file, call `defineIndicator` with a `compute` block, run the
compiler, and render the output through the reference Canvas2D adapter.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the Phase 1 walking-skeleton PR.
