# Inputs

> **Phase:** Lands in Phase 4.
> **Cross-reference:** See PLAN.md §12.

The input-primitives surface that lets a script declare user-tunable
parameters (`input.int`, `input.float`, `input.bool`, `input.string`,
`input.source`, `input.color`, `input.timeframe`, etc.) and the
`@uiHint` JSDoc tag that adapters use to render an input form.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the Phase 4 inputs PR.
