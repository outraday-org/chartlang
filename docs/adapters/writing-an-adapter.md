# Writing an adapter

> **Phase:** Lands in Phase 1.
> **Cross-reference:** See PLAN.md §17.4.

End-to-end guide for adapter authors: scaffold a new adapter with
`@invinite-org/chartlang-adapter-kit`, declare your `Capabilities`,
implement Plot / Draw / Alert handlers, wire a candle stream, and
publish the package to npm. Cross-links to the conformance suite.

Stubbed during the Phase 0 bootstrap so the docs gate has a stable
target. Content lands with the Phase 1 adapter-kit PR.
